# IBM-CUSTOMER-CHURN-PREDICTION

Best Model: Logistic Regression

Accuracy: 82%

Cross Validation Score: 80%

Key Insights:

1. Customers with lower tenure are more likely to churn.
2. Customers with higher monthly charges are more likely to churn.
3. Month-to-month contract customers show higher churn rates.
4. Long-term contracts significantly reduce churn.
